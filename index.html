Server Cleaner Bot

A professional, manual moderation assistant for Discord servers. Built to detect suspicious and bot-like behavior using explainable, confidence-based analysis — without automatic punishments.

Purpose

Server Cleaner Bot helps Discord server owners identify suspicious, bot-like, or alt accounts during raids or spam waves. It provides structured analysis and clear reasoning, while keeping all final decisions in human hands.

Design Philosophy

  • No automatic bans or kicks
  • Owner-only moderation commands
  • Behavior over appearance
  • Explainable results, not guesses
  • Safety over aggression

Detection Signals

  • No roles
  • New account
  • Recently joined
  • Bot-like username
  • No profile picture
  • Generic username patterns
  • Spamming identical messages
  • High message rate
  • Mass mentioning users
  • Likely alt-account logic
  • Message cache analysis (30 messages)
  • Time-based activity patterns

Confidence System

Each signal contributes a weighted score. The combined score produces a confidence level, allowing owners to separate weak indicators from strong behavioral evidence.

  • Low: Minor or informational signals
  • Medium: Multiple suspicious indicators
  • High: Strong, correlated bot-like behavior

Commands & Controls

  • Owner /scan — full member analysis
  • Owner /statscan — server risk overview
  • Owner Kick, Ban, Mute, Quarantine actions
  • Premium Detailed reason inspection

Moderation Safety

The bot ignores administrators and bots, never acts automatically, and always requires explicit confirmation before moderation actions. This prevents abuse, mistakes, and mass false positives.
